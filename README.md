# API Price Shoes

un proyecto personal para practicar la creacion de API desde cero con la arquitectura MVC.
se creo la base de datos en MySQL desde cero 
en el modelo se realizan las consultas a la base de datos
en el controlador se se reciben las peticiones y se envian a la vista ya los datos ordenados des de la consulta de MySQL
ya en la vista se mandan los datos ordenados al front

## Tecnologías Utilizadas
- Node.js
- Express.js
- MySQL
- Versel
- Zod
-GitHub

## Endpoints Disponibles
| Método | Ruta               | Descripción                           |
|--------|--------------------|---------------------------------------|
| GET    | `/api/items`       | Obtiene todos los elementos           |
| POST   | `/api/items`       | Crea un nuevo elemento                |
| PUT    | `/api/items/:id`   | Actualiza un elemento por su ID       |
| DELETE | `/api/items/:id`   | Elimina un elemento por su ID         |
